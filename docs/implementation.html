Implementation Guide

Deploy AEGIS to GitHub Pages in minutes

🚀 Quick Start

This demo is designed to run as a static site on GitHub Pages. No backend required!

📦 Step 1: Create Repository

Create repo
Clone locally
Add files
Push to GitHub
mkdir aegis-demo
cd aegis-demo
git init
git branch -M main
View complete folder structure 
aegis-demo/
├── docs/
│   ├── index.html
│   ├── framework.html
│   ├── sandbox.html
│   ├── technical.html
│   ├── implementation.html
│   └── assets/
│       ├── css/
│       │   └── custom.css
│       └── js/
│           └── app.js
├── README.md
└── .gitignore

📝 Step 2: Add Your Files

Copy all your HTML, CSS, and JavaScript files into the appropriate folders as shown in the structure above.

git add .
git commit -m "Initial AEGIS demo deployment"
git remote add origin https://github.com/YOUR_USERNAME/aegis-demo.git
git push -u origin main
Don't forget to replace YOUR_USERNAME with your actual GitHub username!

🌐 Step 3: Enable GitHub Pages

Go to your repository on GitHub

Navigate to: Settings → Pages
  1. Click on Settings tab in your repository
  2. Navigate to Pages in the left sidebar
  3. Under "Source", select:
    • Branch: main
    • Folder: /docs
  4. Click Save
  5. Wait 2-3 minutes for deployment

Your site will be live at:

https://YOUR_USERNAME.github.io/aegis-demo/

🔧 Troubleshooting

Site not loading after deployment

Try these steps:

  • Wait 3-5 minutes for initial deployment
  • Clear your browser cache
  • Check that /docs folder is selected in GitHub Pages settings
  • Verify index.html exists in the /docs folder
Buttons not working

Common causes:

  • JavaScript file (app.js) not loading properly
  • Linked HTML pages don't exist (404 errors)
  • Check browser console for errors (F12)
  • Verify all file paths are correct
Styling looks broken

Check these items:

  • Verify Tailwind CSS and DaisyUI CDN links are correct
  • Check that custom.css path is correct
  • Ensure data-theme="dark" is set on html tag
  • Clear browser cache and hard refresh (Ctrl+Shift+R)
